# 本地真机测试断点 051

时间：2026-07-10 17:55

## 问题

企业微信使用 `Alt+Shift+A` 截图后，微软拼音会变成英文模式。

## 根本原因

- 侧边栏截图监听只读取剪贴板图片并展示确认提示，没有调用 `focus()`、键盘事件或输入法切换接口。
- 企业微信截图快捷键包含 `Shift`。
- 微软拼音原本同时启用了 `Ctrl + 空格键` 和 `Shift` 两种中/英文模式切换方式。
- 因此按下 `Alt+Shift+A` 时，Shift 会触发微软拼音的中/英文模式切换。

## 已修复

- 打开 Windows 设置 -> 时间和语言 -> 语言和区域 -> 微软拼音 -> 按键。
- 保留 `Ctrl + 空格键` 中/英文切换。
- 关闭 `Shift` 中/英文切换。
- 系统已持久化：`HKCU\\Software\\Microsoft\\InputMethod\\Settings\\CHS` 下 `English Switch Key = 2`。

## 真机验证

- 企业微信聊天输入框获得焦点。
- 截图前任务栏显示 `中文模式`。
- 使用 `Alt+Shift+A` 进入企业微信截图。
- 拖选区域并按 Enter 完成截图。
- 截图完成回到企业微信后，任务栏仍显示 `中文模式`。
- 未向聊天会话发送任何消息。

## 回退方法

如需恢复 Shift 切换：Windows 设置 -> 微软拼音 -> 按键 -> 重新开启 `Shift`。
